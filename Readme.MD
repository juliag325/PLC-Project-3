**How to Run Our Code**
Make sure cabal is installed and updated. 

*cabal build*

To Run Tests: 
    1. Pascal test cases: 
        *cabal run tests/test4.pas*
        (Replace test4.pas with any test needed)
    2. Hspec unit tests: 
        *runhaskell InterpretSpec.hs* or 
        *runhaskell ValSpec.hs*
        etc.
